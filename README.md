# Data Analysis with Python - IBM Course

This repository contains all the code and projects I completed while taking the **Data Analysis with Python** course by IBM on Coursera.

## Course Details

- **Course Provider:** IBM
- **Platform:** Coursera
- **Course Description:** This course covers the basics of data analysis with Python, including data wrangling, data visualization, and machine learning using libraries such as pandas, numpy, matplotlib, seaborn, and scikit-learn.

## Course Content

The repository is organized into the following sections:

1. **Week 1:** Introduction to Data Analysis with Python
2. **Week 2:** Data Wrangling
3. **Week 3:** Exploratory Data Analysis
4. **Week 4:** Model Development
5. **Week 5:** Model Evaluation
